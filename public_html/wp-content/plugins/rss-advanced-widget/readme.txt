== Description ==

This plugin create a widget to set the reading of RSS. It works with both RSS and XML. There are a few parameters to set up: Title = The title to display in your blog, RSS Feed = The URL of your selected RSS (work both rss or xml feed), Item to display = how many item display (from 1 to 5), show date = y/n, show summary = y/n.



== Installation ==


1. Upload the zip file of the plugin to the '/wp-content/plugins/' directory.

2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the Appearence/Widget section of your Wordpress and insert plugin.
